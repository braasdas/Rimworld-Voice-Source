# Frontend Source Files

The C# source files are located in the original repository at:
`ColonistVoices-Backend/Source/`

## Files to Copy:
- AudioManager.cs
- BackendAPIHandler.cs
- ColonistContextBuilder.cs
- ColonistVoices.csproj
- ColonistVoicesSettings.cs
- CoroutineManager.cs
- SpeechController.cs
- SpeechHistoryEntry.cs
- VoiceSelector.cs

## Build Instructions:
1. Install Visual Studio 2019+ or .NET Framework SDK
2. Run build_simple.bat (Windows) or use msbuild
3. DLL will be output to Assemblies folder

See [../docs/FRONTEND_SETUP.md](../docs/FRONTEND_SETUP.md) for detailed instructions.
